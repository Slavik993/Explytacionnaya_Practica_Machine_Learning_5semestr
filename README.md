# Прогнозирование потребности в кадровых ресурсах с помощью нейронных сетей и стекинга



**Автор**: Шебанов Вячеслав  

**Вариант**: 15  

**Тема**: Применение нейронных сетей для прогнозирования потребности в кадровых ресурсах  

**Датасет**: `WA_Fn-UseC_-HR-Employee-Attrition.csv` (IBM HR Analytics)
**Временные ряды**: синтезированы на основе датасета  



## Результаты

**Лучшая модель**: Стекинг (Bi-LSTM + Attention + XGBoost + LightGBM + Ridge)
**Метрика MAPE на тесте**: **~7.1–8.5%** (в зависимости от сида)

Прогноз на 2025–2026 гг.

Все графики, модели и данные сохранены в папке `artifacts/`

## Инструкция по запуску

В коде есть markdown комментарии описывающие блоки кода, для работы нужно использовать библиотеки numpy pandas matplotlib seaborn scikit-learn tensorflow xgboost lightgbm openpyxl

Для корректной работы запускать по одному блоку последовательно от начала до конца.
